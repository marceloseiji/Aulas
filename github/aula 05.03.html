Interative Content

onteúdo que permite a interação do usuário Nota: Alguns elementos no HTML podem ser ativados por um comportamento. Isso significa que o usuário pode ativá-lo de alguma forma. O início da sequência de eventos depende de mecanismo de ativação e normalmente culminam em um evento de click seguido pelo evento DCMativate. O user agent permite que o usuário ative manualmente o elemento que tem este comportamento utilizando um teclado, mouse, comando de voz, etc.

  • a
  • audio (se o atributo control for utilizado)
  • button
  • details
  • embed
  • iframe
  • img (se o atributo use map for utilizado)
  • input (se o atributo type não tiver o valor hidden)
  • keygen
  • label
  • menu(se o atributo type tiver o valor toolbar
  • select
  • textarea
  • video (se o atributo control for utilizado
